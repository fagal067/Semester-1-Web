VB Command List

Dim

Tells computer to label and keep track of a variable
- Example: Dim num as Integer


If/then

Allows computer to make a decision depending on a scenario
- Example: If num > 17 then [...]


Select Case

Allows computer to make a decision with many options
- Example:
select Choice
Case 1
Phone = "Pear Phone 15"
Cost = 2000

Case 2
Phone = "Pear Phone 15 Plus"
Cost = 2500

Case 3
Phone = "Pear Phone 20"
Cost = 10000
end select

For Loop

Tells computer to repeat something for a specific amount of times
- Example: for count as Integer = 1 to 30
console.Writeline(count)

Input

Has computer read an input from the user
- Example: console.Readline()

Output

Has computer print a string, object, variable, or integer
- Example: console.WriteLine("Hello, world")
